# Examples

This folder contains small report examples for Maya Production Pipeliner.

Current files:

```text
example_report.txt
example_report.json
```

The current examples show the initial safe Dry Run smoke output generated outside Maya. They demonstrate report shape, summary fields, hook status, and empty route decisions when no Maya scene runtime is available.

Live Maya examples should only replace these files after they are generated by the tool from a real test scene.
